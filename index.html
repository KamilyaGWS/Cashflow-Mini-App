Инструкция
Руководство для администратора
🚀 Быстрый старт ▾
1
Перейдите на вкладку Управление — она видна только администратору.
2
Сначала добавьте всех участников через блок Добавить участника. Укажите имя и попросите каждого написать боту @userinfobot чтобы узнать свой Telegram ID.
3
После каждой игры открывайте Записать игру, выбирайте дату, добавляйте участников и вводите их место и FIQ.
4
Нажмите 💾 Сохранить игру. Печати начислятся автоматически: 1 место = 3 🔖, 2 место = 2 🔖, 3 место = 1 🔖.
5
При накоплении 9 печатей игрок автоматически получает 🎫 бесплатный билет на игру.
📱 Что делает каждая вкладка ▾
1
Мои игры — каждый участник видит свою статистику: печати, билеты, средний и максимальный FIQ, прогресс к следующему билету. Отображается только при открытии через Telegram.
2
Рейтинг FIQ — рейтинг всех участников по среднему FIQ. Чем выше среднее значение Financial IQ — тем выше место.
3
Статистика — общий рейтинг по накопленным печатям. Здесь видно кто лидирует по количеству побед.
4
Управление (только для администратора) — добавление участников, редактирование данных, запись новых игр, использование билетов.
5
История игр (только для администратора) — все записанные игры сгруппированы по датам. Можно просматривать, а при необходимости серьёзные исправления делать через Google Sheets.
👤 Как добавить участника ▾
1
Откройте Управление → Добавить участника.
2
Введите полное имя участника так, как оно будет отображаться в рейтинге.
3
Попросите участника написать в Telegram боту @userinfobot — он пришлёт числовой Telegram ID. Вставьте это число в поле Telegram ID.
4
Нажмите + Добавить участника. Система проверит дубликаты автоматически.
⚠️ Если вы неправильно добавили участника, исправить имя можно через блок Управление участником. Удалить участника можно только через Google Sheets.
🎮 Как записать игру ▾
1
Откройте Управление → Записать игру.
2
Выберите дату игры. Помните: одна дата = одна игра. Нельзя записать две игры на один день.
3
Нажмите + Добавить участника для каждого игрока. Выберите имя из выпадающего списка.
4
Для каждого участника укажите место (1, 2, 3 или 4+) и значение FIQ из карточки игры.
5
Нажмите 💾 Сохранить игру.
⚠️ Если вы случайно ввели неверные данные, исправить их можно только в Google Sheets — см. раздел «Исправление ошибок» ниже.
FIQ — это числовое значение Financial IQ из карточки участника после завершения игры Cashflow. Может быть целым (1800) или дробным (2100.56).
🔖 Как работают печати и билеты ▾
1
Печати начисляются автоматически после каждой игры:
🥇 1 место = 3 печати
🥈 2 место = 2 печати
🥉 3 место = 1 печать
4+ место = 0 печатей
2
Когда у участника накапливается 9 печатей — система автоматически выдаёт ему 🎫 1 бесплатный билет на следующую игру.
3
Чтобы использовать билет, откройте Управление → Управление участником, выберите игрока и нажмите 🎫 Использовать билет.
⚠️ Использование билета нельзя отменить через приложение. Если нажали по ошибке — обратитесь к Google Sheets и вручную увеличьте число в столбце «tickets».
🛠️ Исправление ошибок через Google Sheets ▾
⚠️ Если данные внесены неверно (неправильный FIQ, место, имя), исправить их можно только напрямую в Google Sheets. Приложение считывает данные из таблицы в реальном времени.
📊 Открыть Google Sheets →
1
Лист Players — данные всех участников. Столбцы: id, full_name, telegram_id, total_stamps, used_stamps, games_played, tickets, created_at. Можно менять имя и telegram_id вручную.
2
Лист FIQ_History — история всех результатов. Найдите нужную строку по game_id и player_id, исправьте значения place и fiq. Не удаляйте строки без необходимости — это повлияет на статистику.
3
Лист Games — список игр. Каждая строка = одна игра с уникальным game_id.
4
Как найти нужную строку: используйте Ctrl+F (или ⌘+F на Mac) для поиска по тексту в таблице. Введите имя участника или дату.
5
После исправления данных в Google Sheets обновите приложение — данные подтянутся автоматически при следующем открытии вкладки.
⚠️ Удаление данных: удалять строки из Players или FIQ_History нужно осторожно. Если удалить игрока из Players — его история в FIQ_History останется, но не будет привязана к профилю. Рекомендуем не удалять, а очищать поля telegram_id если участник больше не играет.
❓ Частые вопросы ▾
Участник не видит свою статистику в «Мои игры»?
Значит его Telegram ID не добавлен или добавлен неверно. Попросите написать @userinfobot и передать вам ID. Добавьте через Управление → Управление участником.
Бот не отвечает?
Telegram-бот иногда работает с задержкой до 10 секунд. Если бот не отвечает более 30 секунд — проверьте Mini App: откройте его напрямую через кнопку в боте. Данные в Mini App загружаются независимо от бота.
Приложение показывает «Ошибка загрузки»?
Закройте и откройте заново. Если ошибка повторяется — проверьте Google Sheets что данные на месте. Обратитесь к таблице.
Как добавить исторические данные за прошлые игры?
Используйте блок Записать игру и выберите прошедшую дату. Либо добавьте строки напрямую в лист FIQ_History в Google Sheets. Формат: game_id, date, player_id, player_name, place, fiq, stamps.
